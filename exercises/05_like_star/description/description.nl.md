# Oefening 5 — Films met 'Star' in de titel

Je werkt op de databank **films.sqlite**.  
Deze bevat één tabel **Films** met volgende kolommen:

| Kolom | Omschrijving |
|-------|---------------|
| FilmID | Uniek nummer van de film |
| Titel | Naam van de film |
| Genre | Soort film |
| Regisseur | Naam van de regisseur |
| Jaar | Verschijningsjaar |
| Duur | Duur in minuten |
| Taal | Originele taal |
| Land | Land van herkomst |
| Rating | Beoordeling (0–10) |
| Beschikbaar | 1 = beschikbaar, 0 = niet beschikbaar |

---

## Opdracht

Toon **alle films waarvan de titel het woord 'Star' bevat**.  
Geef de kolommen: `Titel`, `Regisseur`, `Jaar`, `Rating`.
